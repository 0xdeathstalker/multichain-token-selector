## Getting Started

```
# git clone
git clone git@github.com:0xdeathstalker/multichain-token-selector.git

# create .env file
cp .env.example .env

# install dependencies
pnpm install

# run the development server
pnpm dev
```

### Getting the API key

You can get the API key from [Dune](https://dune.com/settings/api).
